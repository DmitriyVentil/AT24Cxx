# AT24Cxx
	Библиотека работает со следующими микросхемами: 
- 24C00,
- AT24C1,
- AT24C2,
- AT24C4,
- AT24C8,
- AT24C16,
- AT24C32,
- AT24C64,
- AT24C128,
- AT24C256,
- AT24C512,
- AT24CM01,
- AT24CM02.

	Библиотека позволяет записывать, читать и удалять, начиная с любого адреса и заканчивая любым адресом памяти, без привязки к страницам самой микросхемы.
